# Media Document Test Checklist

1. iOS sends JPG to Android.
2. iOS sends HEIC image to Android.
3. Android sends image to iOS.
4. iOS sends PDF from local Files.
5. iOS sends PDF from iCloud Drive.
6. iOS sends XLSX.
7. iOS sends DOCX.
8. iOS sends CSV.
9. iOS sends TXT.
10. iOS sends ZIP.
11. Android sends PDF/XLSX/DOCX to iOS.
12. Filename remains correct.
13. Extension remains correct.
14. File size displays correctly.
15. PDF previews in app.
16. XLSX/DOCX use Quick Look or Open In fallback.
17. Ending preview returns safely to chat.
18. Text and calls still work afterward.
